# hermisJS - JavaScript Programming Laboratory

Welcome to hermisJS, a comprehensive JavaScript programming laboratory featuring 20 progressive programming exercises designed to build fundamental JavaScript skills.

## Overview

This project contains a comprehensive series of JavaScript programs organized into three phases:
- **Phase 1:** Fundamentals (20 exercises covering basic syntax to functions)
- **Phase 2:** Intermediate Concepts (20 exercises covering functions, OOP, and advanced object manipulation)
- **Phase 3:** Advanced Concepts (20 exercises covering array methods, DOM manipulation, strings, regex, JSON, and error handling)

Each program is self-contained with a verification header for tracking execution and student information.

## Running the Programs

### In a Browser
Open `index.html` in your web browser to run the current program (defaults to prog-20.js). To run a different program, modify the script source in `index.html`.

### In Node.js
Run any program directly using Node.js:
```bash
# Phase 1 Programs
node phase-1/prog-01.js
node phase-1/prog-10.js
node phase-1/prog-20.js

# Phase 2 Programs
node phase-2/prog-01.js
node phase-2/prog-15.js
node phase-2/prog-20.js
```

### In Browser Console
Open your browser's developer tools (F12) and execute:
```javascript
// Load and execute a program
fetch('phase-1/prog-01.js')
  .then(response => response.text())
  .then(code => eval(code))
```

## Key Concepts Covered

### Phase 1
- **Core JavaScript:** Variables, data types, operators, functions
- **Control Flow:** Conditionals (if/else/switch), loops (for/while)
- **Data Structures:** Arrays, objects, strings
- **Built-in Objects:** Math, Date, Array, String
- **User Interaction:** Console output, prompt input
- **Functions:** Declaration, invocation, parameters, return values

### Phase 2
- **Function Paradigms:** Function declarations, expressions, arrow functions, closures
- **Advanced Functions:** Higher-order functions, callbacks, recursion, IIFE
- **Context Control:** Understanding 'this', call(), apply(), bind()
- **Object Creation:** Literals, constructors, classes
- **Inheritance Patterns:** Prototypal inheritance, class-based inheritance, Object.create()
- **Property Descriptors:** Getters, setters, property control
- **Object Methods:** Object.keys(), Object.entries(), Object.assign(), Object.is()
- **OOP Concepts:** Encapsulation, private fields, static methods

### Phase 3
- **Array Methods:** Creation, iteration, transformation, search, modification, sorting
- **Array Techniques:** Spread operator, destructuring, multidimensional arrays
- **Array Utilities:** Array.from(), Array.of(), array-like objects
- **DOM Selection:** getElementById, getElementsByClassName, querySelector, querySelectorAll
- **DOM Manipulation:** textContent, innerHTML, innerText, element creation, removal, cloning
- **DOM Attributes:** setAttribute, getAttribute, data attributes, classList
- **DOM Events:** Event listeners, mouse events, keyboard events, form events
- **Event Handling:** Event bubbling, capturing, delegation, preventDefault, stopPropagation
- **String Methods:** Case, search, extraction, replacement, padding, repetition
- **Regular Expressions:** Pattern matching, flags, character classes, quantifiers, validation
- **JSON:** Serialization, deserialization, object conversion
- **Error Handling:** Try-catch-finally, error types, custom errors, debugging techniques

## Learning Outcomes

### Phase 1
Upon completing Phase 1, students will be able to:
- Write and execute JavaScript programs
- Understand variables and data types
- Control program flow with conditionals and loops
- Work with arrays and objects
- Manipulate strings and dates
- Declare and invoke functions
- Use console for debugging and output

Upon completing Phase 2, students will be able to:

- Understand and control function context ('this' binding)

- Build more complex object hierarchies and applications

### Phase 3
Upon completing Phase 3, students will be able to:
- Master all built-in array methods for data processing
- Use spread operators and destructuring for elegant code
- Query and manipulate the DOM programmatically
- Handle DOM events and event delegation patterns
- Process and manipulate strings efficiently
- Use regular expressions for pattern matching and validation
- Serialize and deserialize JSON data
- Implement proper error handling and debugging techniques
- Build interactive web applications with DOM and event handling
